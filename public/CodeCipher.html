CodeCipher
Note: Kindly refer the schedule before registering events
DESCRIPTION

An exciting coding challenge where students decode logic-based questions, write the correct piece of code, and determine the final output. Test your analytical thinking, coding accuracy, and debugging skills in this fast-paced contest!

RULES
    1. Each problem will be presented with a unique clue.
    2. The task is to find and write the correct output only.
    3. Answers must be submitted within the given time limit.
    4. Use of external compilers, internet, or AI tools is strictly prohibited.
    5. The participant with the highest score in the least time will be declared the winner.
